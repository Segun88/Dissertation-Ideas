# Replication Files for *Rebel Groups and Human Rights Violations*

## Overview
This repository replicates the analysis for *Rebel Groups and Human Rights Violations*, examining how rebel origins shape patterns of human rights abuses.

## Folder Structure
- `data/raw/` — original input data
- `data/processed/` — cleaned/merged datasets created by scripts
- `R/` — R scripts
- 
## Data Sources
- FORGE (founding organizations)
- Human Rights Violations datasets (e.g., RHRV)
- Non-State Actors datasets (e.g., NSA)
